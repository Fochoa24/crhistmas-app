# Gift List Application

Aplicación para gestionar listas de regalos familiares.

## Características
- Gestión de integrantes (Agregar/Eliminar)
- Listas de regalos por integrante
- Persistencia de datos

## Cómo correr la aplicación

1. Instalar dependencias:
```bash
npm install
```

2. Correr servidor de desarrollo:
```bash
npm run dev
```

## Verificación

Se ha verificado el funcionamiento de:
- Agregar/Eliminar padres
- Agregar/Eliminar regalos
- Persistencia local
